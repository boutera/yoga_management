# Yoga Management System

A yoga studio management system built with React, Node.js, and MongoDB.

## Prerequisites

- Node.js (v14+)
- MongoDB (v4.4+)
- npm (v6+)

## Quick Start

### Frontend Setup

```bash
npm install
```

 Run the frontend:
```bash
npm run dev
```

### Backend Setup

1. Navigate to backend directory and install dependencies:
```bash
cd backend
npm install
```

2. Start MongoDB:
```bash
# Windows
net start MongoDB
```

3. Run the backend:
```bash
npm start
```



## Access Points

- Frontend: http://localhost:5173
- Backend API: http://localhost:5000
